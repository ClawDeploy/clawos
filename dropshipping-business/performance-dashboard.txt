## OPTIMIZATION PLAYBOOK

### When ROAS is Low (<2x):
1. Pause underperforming ad sets
2. Check frequency (if >3, refresh creative)
3. Narrow audience targeting
4. Increase creative testing
5. Review landing page conversion

### When CPA is High (>$20):
1. Test new audiences
2. Improve ad creative/offer
3. Add retargeting campaigns
4. Optimize for add-to-cart first
5. Check for bot traffic

### When Conversion Rate is Low (<2%):
1. Add trust badges
2. Improve product page images
3. Add social proof (reviews)
4. Simplify checkout process
5. Add urgency elements
6. Test different pricing

### When AOV is Low (<$30):
1. Introduce bundles
2. Add post-purchase upsells
3. Set free shipping threshold
4. Create "complete the look" recommendations
5. Offer volume discounts

### When Email Open Rates are Low (<15%):
1. Test new subject lines
2. Clean email list
3. Adjust send times
4. Segment audience
5. Personalize sender name
